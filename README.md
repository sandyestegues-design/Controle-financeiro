# Controle-financeiro Pessoal
Aplicativo simples de controle financeiro pessoal
## Funcionalidades
- Resgistrar despesas e receitas
- Calcular saldo atual;
- Listar histórico de transações;
- Visualizar total de gastos por cate
- Possibilidade de salvar registros 1

- ## Tecnologias utilizadas
  - Python 3.x
  - Manipulção de arquivos (CSV OU JSO)
  - Git e Github para versionamento
